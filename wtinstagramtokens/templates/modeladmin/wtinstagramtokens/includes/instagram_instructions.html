1. Login to the Instagram.
  2. Open the https://www.instagram.com/developer/clients/manage/ and register new client.
  3. Fill out the fields. Set the redirect URL to the http://{{ request.META.HTTP_HOST }}/instagram_config/ (please make sure there are no spaces there).
  4. Save the client and open it again and copy Client ID and Secret.
  5. Open the Admin/Settings/Instagram Tokens and set the Client ID and Client Secret there. Please make sure there are no any spaces there.
  6. Click the Save button and wait for the Instagram security settings dialog for this App. You MUST confirm access to your Instagram feeds.
  7. When wait for a little bit and you will see the "Success" status there.
